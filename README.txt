CEDA IPCC DDC climatologies for CRU TS3.24.01
=============================================

This repository contains the files used to calculate the climatologies 
of the variables in CRU TS3.24.01 / used in AR5 for the IPCC DDC 
(data distribution centre).

Also contained is code to produce a GeoTIFF from a netCDF file, and the
required color tables.

Requires:
    - CDO (climate data operators)
    - NCO (netCDF operators)
    - Python2.7
    - GDAL (Geospatial Data Abstraction Library) - for creating the GeoTIFFs
